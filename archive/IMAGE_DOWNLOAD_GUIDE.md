# Image Download Guide

This file contains all external image and video URLs that need to be downloaded and saved locally.

## Directory Structure

The following directories have been created:

- `public/images/hero/` - Hero section images
- `public/images/gallery/` - Gallery section images
- `public/images/features/` - Feature section images
- `public/images/placeholders/` - Placeholder images
- `public/images/products/` - Product images for Gallery24
- `public/images/case-studies/` - Case study background images for Feature222
- `public/videos/hero/` - Hero section videos
- `public/videos/gallery/` - Gallery section videos

## Download Instructions

You can use these commands to download all images:

```bash
# Hero images
cd public/images/hero
curl -o hero-1.png "https://framerusercontent.com/images/PkU3Gw8pAzmi2niiMrnBcHuu8I.png"
curl -o hero-2.jpg "https://framerusercontent.com/images/IIwrTUEKmVA8Bc0vqQYJIoYqps.jpg"
# ... (continue for all images)

# Or use a download script
```

After downloading, delete this file as it's no longer needed.
